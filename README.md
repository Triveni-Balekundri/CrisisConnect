# CrisisConnect - Community Crisis Support Platform

CrisisConnect is a responsive web platform that connects people in emergency situations with nearby volunteers for faster community support.
Currently we have a Prototype demo video included in this repo.

## Problem Statement
During floods, accidents, fires, or local emergencies, urgent requests often fail to reach nearby people in time.  
CrisisConnect helps improve response by making emergency requests visible to trusted volunteers in the local area.

## Features
- Emergency help request form
- Category-based support (food, medical, shelter, transport)
- Location-based request tracking
- Volunteer response dashboard
- Request status updates
- Admin monitoring

## Tech Stack
- React.js
- HTML CSS
- Firebase (DB|Auth)
- Google Maps API
- GitHub 

## Project Workflow
1. User submits emergency request  
2. Request is stored in database  
3. Nearby volunteers receive request  
4. Volunteer responds  
5. Admin monitors activity
